# DSA Problems in Python

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

## Running Solutions

### Individual files:
```bash
python filename.py
```

### With pytest:
```bash
pytest filename.py
pytest  # Run all tests
```

## Template Usage

Use `template.py` as a starting point for new problems. Copy and rename it for each new problem.

## Structure

- Each solution file should be self-contained
- Include test cases in the same file
- Use descriptive filenames (e.g., `two_sum.py`, `binary_search.py`)
